## Project Overview

**Bookshelf App** is a simple and interactive front-end web application
for managing a personal book collection.\
Users can add books, update their reading status, edit details, delete
entries, and search through the list efficiently.

The app stores all data using **Web Storage (localStorage)**, ensuring
the bookshelf remains available even after the browser is closed.
This project highlights core front-end concepts such as **DOM
manipulation, event handling, UI feedback using modal dialogs, and
client-side data persistence**.

## Objectives

This project was created as part of the Dicoding Front-End Web
Developer Learning Path.

The primary objective is to build a fully functional bookshelf
management app that enables users to:

-  Manage book data through a user-friendly interface
-  Organize books into "Finished" and "Unfinished" categories
-  Modify book information
-  Maintain data persistently using localStorage

## Features

-  **Add Books**: Insert new books with title, author, publication year, and reading status
-  **Edit Books**: Update book information
-  **Delete Books**: Remove books from bookshelf.
-  **Move Between Shelves**: Toggle reading status (finished/unfinished)
-  **Search Function**: Find books by title

## Tech Stack

-  HTML5
-  CSS3
-  JavaScript (ES6+)
-  DOM Manipulation
-  Event Handling
-  Web Storage API (localStorage)
-  SweetAlert2
-  Feather Icons

## Installation & Usage

#### 1. Clone Repository

```bash
git clone https://github.com/kellychan3/Bookshelf-App.git
cd Bookshelf-App
```

#### 2. Run the project

Simply open **index.html** on your browser. No additional setup, frameworks, or build tools are required.
